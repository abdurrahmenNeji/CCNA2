# CCNA 2 Networking Labs – Routing and Switching

## 📌 Overview

This repository contains my **CCNA 2 networking labs** completed during my studies in the **Professional Master SSI at ISG Tunis**.

The labs focus on practical network configuration using **Cisco Packet Tracer**, including VLANs, trunking, STP, static routing, default routing, IPv6 routing, and troubleshooting.

This repository is part of my learning path in networking, cybersecurity, and system administration.

---

## 🎯 Objectives

The main objectives of this repository are to:

- Practice Cisco networking concepts through real lab scenarios
- Configure routers and switches using Cisco IOS commands
- Understand VLANs and trunk links
- Configure and troubleshoot STP
- Configure IPv4 and IPv6 static routes
- Configure default and floating static routes
- Test network connectivity using Packet Tracer
- Document each lab with reports and configuration details

---

## 🛠️ Tools Used

- Cisco Packet Tracer
- Cisco IOS CLI
- Routers
- Switches
- PCs and end devices
- Microsoft Word for lab reports
